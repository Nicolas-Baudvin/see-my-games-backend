# SEE MY GAMES BACKEND - NODE JS

## Description

Code source du back end pour l'application web See My Games

## Install

```bash
    git clone https://github.com/Nicolas-Baudvin/see-my-games-backend.git
```

```bash
    yarn install # or npm
```

```bash
    yarn start # or npm
```
